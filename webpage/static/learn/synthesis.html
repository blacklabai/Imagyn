Make more training data by transforming existing images

Transformations

Using various software packages, we modify the images that are collected by applying various transformations. Transformations include adding noise, rotating, skewing, and carving to name a few. Transformations are applied to help the neural network that will train on these images recognize occurrences of them when they're not represented as nearly as in other training data.

Imagyn allows you to choose the input parameters for any transformations you select, or it can abstract those away and generate input parameters for you. For example, take rotation - this transformation would take one extra parameter, the number of degrees to rotate by. You can supply this if you'd like, otherwise Imagyn will fill it for you. If you choose to let Imagyn select a parameter, it will do so smartly, knowing that a rotation of 1 degree isn't going to change anything much. Imagyn uses statistical distributions to select parameters, for rotation this is done using a normal gaussian distributions to select rotation degrees between 0-90 degrees as anything. The same principle is applied to other transformations such as brightness, contrast, and padding.

Contrast

Differing contrast helps image recognizers when the input is a photo - contrast isn't going to be the same every time for every photo.

Noise

Some image recognizers completly fail when a noise layer is added over input images, adding noise before training helps them perform better in these cases.

Rotation

Rotation helps image recognizers learn that the object they're looking for isn't always orientated the same way.

Hue shifting

This transformation has less broad use cases, than the above, but some Instagram filters filters for example perform hue shifting on a lower scale than the image represented here.

Brightness

Changing brightness has a similar effect as changing contrast - it usually helps guide recognizers to learn more strongly about the shape of the object than its colors.

Mirroring

This can also be achieved with rotation. Synthesized mirroring helps in exactly the same way as rotation does. Imagyn provides horizontal, vertical, and combined "diagonal" mirroring.

Grayscale

Especially useful for tasks where input data comes from social media such as Instagram and Snapchat, as these platforms have embedded grayscale filters.

Padding

Most image recognition neural networks resize their images to a standard size to match the number of input neurons, effectively nullifying any gains from performing scaling operations. By padding an image a neural recognizer can learn about scale.

Seam Carving

Seam carving removes a connected seam of pixels with the least amount of 'energy' across one dimension of an image. This operation provides the ability to scale images without scaling the subject.

Skew

Skewing images allows the image recognizer to recognize photos of objects that were not taken front on.

Sharpening

Both sharpening and softening are techniques used in tandem to increase edge detection in neural recognizers.

Softening

Softening images can be used - alongside sharpening - to coax image recognizers to learn about edges.
