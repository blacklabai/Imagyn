Learn More / Synthesis

Make more training data by transforming existing images

Transformations

Using various software packages, we modify the images that are collected by applying various transformations. Transformations include adding noise, rotating, skewing, and carving to name a few. Transformations are applied to help the neural network that will train on these images recognize occurrences of them when they're not represented as nearly as in other training data.

Imagyn allows you to choose the input parameters for any transformations you select, or it can abstract those away and generate input parameters for you. For example, take rotation - this transformation would take one extra parameter, the number of degrees to rotate by. You can supply this if you'd like, otherwise Imagyn will fill it for you. If you choose to let Imagyn select a parameter, it will do so smartly, knowing that a rotation of 1 degree isn't going to change anything much. Imagyn uses statistical distributions to select parameters, for rotation this is done using a normal gaussian distributions to select rotation degrees between 0-90 degrees as anything. The same principle is applied to other transformations such as brightness, contrast, and padding.
