About the image downloader and related interfaces

Downloading images

We use ImageNet as our source of annotated images. ImageNet is based off of a library called WordNet: a "large lexical database of English. Nouns, verbs, adjectives and adverbs are grouped into sets of cognitive synonyms (synsets), each expressing a distinct concept". ImageNet simply maps images to each synset in WordNet. We then retrieve the images from ImageNet's API.

ImageNet annotates images, but doesn't make them available on their website for licensing reasons. ImageNet instead provides a list of urls from where images can be downloaded from. Training a recognizer on copyrighted images is okay, since the protected images are discarded and not republished anywhere. An issue though, is that the internet is dynamic, and a link may break if the image it tracked was removed. To overcome this, our Collections module is best-effort; take for example, the Downloader class: it tries to download as many images as it can and gracefully continues trying if an image cannot be located.

Some websites might still be slow to respond, so the Downloader class leverages parallelism by running the download task in multiple processes. This way we don't get completely hung-up on one slow website and are able to keep downloading the images you've requested. There's several ways to run the downloader:

  • Import the module and call it 

    from imagyn.collections.download import Downloader
    Downloader().multidownload( ... )

  • Run the downloader from the command-line 

    $> ./imagyn/collections/download.py <keyword> --number 10 <output_dir>

Working with words and synsets

The collections module also contains a lexicon for interaction with both WordNet and ImageNet. It provides several utility functions for working with the data sources. The common use case follows the path Keyword -> Synset -> URLs. The lexicon provides the tools to easily step through this using the following functions: 

from imagyn.collection.lexicon import SynsetLexicon
lexicon = SynsetLexicon()

  • Keyword to Synset: synset = lexicon.get_synset(<keyword>)

  • Synset to URLs: urls = lexicon.API.urlsfor(lexicon.get_synset_id(synset))

  • Synset relatives:

    • lexicon.get_parents(synset)

    • lexicon.get_siblings(synset)

    • lexicon.get_grandparents(synset)

  • Confirm ImageNet has entries for a Synset: lexicon.valid_synset(synset)
