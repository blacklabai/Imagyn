Image Retrieval

We use ImageNet as our source of annotated images. ImageNet is based off of a library called WordNet: a "large lexical database of English. Nouns, verbs, adjectives and adverbs are grouped into sets of cognitive synonyms (synsets), each expressing a distinct concept". ImageNet simply maps images to each synset in WordNet. We then retrieve the images from ImageNet's API.

Learn more

Image Synthesis

Using various software packages, we modify the images that are collected by applying various transformations. Transformations include adding noise, rotating, skewing, and carving to name a few. Transformations are applied to help the neural network that will train on these images recognize occurrences of them when they're not represented as nearly as in other training data.

Pipelining and UI

Imagyn uses Spotify's Luigi to manage all of the tasks that are required to run through the process of assembling a new dataset. Luigi helps us configure and address issues typically associated with long-running batch processes.
